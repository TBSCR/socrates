Django Template Project

Use this template to get your project quickly up and running. Many of the features that you would most likely implement anyway are already there:

  • Base template
  • URL configuration
  • Example use of a class-based view
  • Login/logout using Django Auth
  • Django REST Framework
  • Twitter Bootstrap
  • Fonts Awesome
  • Vagrant

Django 1.10 + Python 3

This template is based on Django version 1.10 and Python 3.

Full documentation of Django and a great tutorial here: https://docs.djangoproject.com/en/1.10/

Vagrant

Vagrant is a great tool for development and testing. It enables you to run your project in a virtual Ubuntu environment, regardless of your operating system. In order to use Vagrant, you need to have installed Virtualbox on your computer.

Start Vagrant with command vagrant up

Connect to the vagrant machine with command vagrant ssh and navigate to cd /vagrant to see files shared with the host machine.

Vagrant's configuration is defined in files vagrantfile and bootstrap.sh.

Bootstrap 3

This template uses Twitter Bootstrap 3 as the basic UI framework.

Bootstrap is a HTML, CSS and JS framework, which makes development of responsive and modern layouts super easy.

Full documentation of Bootstrap here: hhttp://getbootstrap.com/
